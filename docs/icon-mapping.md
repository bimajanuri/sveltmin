# Mapping Icon dari Hero Icons ke Flowbite Svelte Icons

Berikut adalah tabel mapping icon yang digunakan di TailAdmin React (Hero Icons) ke padanan Flowbite Svelte Icons:

| TailAdmin React (Hero Icons) | Flowbite Svelte Icons | Catatan |
|------------------------------|-----------------------|---------|
| HomeIcon | HomeSolid atau HomeOutline | Tersedia dalam berbagai varian |
| GridIcon | GridSolid atau GridOutline | Digunakan untuk menu Dashboard |
| CalenderIcon | CalendarSolid atau CalendarOutline | Digunakan untuk menu Calendar |
| UserCircleIcon | UserCircleSolid atau UserCircleOutline | Digunakan untuk menu User Profile |
| ListIcon | ListSolid atau ListOutline | Digunakan untuk menu Forms |
| TableIcon | TableSolid atau TableOutline | Digunakan untuk menu Tables |
| PageIcon | DocumentSolid atau DocumentOutline | Digunakan untuk menu Pages |
| PieChartIcon | ChartPieSolid atau ChartPieOutline | Digunakan untuk menu Charts |
| BoxCubeIcon | CubeSolid atau CubeOutline | Digunakan untuk menu UI Elements |
| PlugInIcon | PlugSolid atau PlugOutline | Digunakan untuk menu Authentication |
| ChevronDownIcon | ChevronDownSolid atau ChevronDownOutline | Untuk indikator dropdown |
| ArrowUpIcon | ArrowUpSolid atau ArrowUpOutline | Untuk indikator kenaikan |
| ArrowDownIcon | ArrowDownSolid atau ArrowDownOutline | Untuk indikator penurunan |
| GroupIcon | UsersSolid atau UsersOutline | Untuk metric Customers |
| BoxIconLine | BoxSolid atau BoxOutline | Untuk metric Orders |

Catatan:
1. Flowbite Svelte Icons menyediakan berbagai varian setiap icon (Solid, Outline, dll.)
2. Pemilihan varian spesifik tergantung pada desain visual TailAdmin
3. Untuk implementasi penuh, perlu memilih varian yang paling sesuai secara visual
